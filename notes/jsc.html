JavaScript – Detailed Notes

1. Introduction to JavaScript

JavaScript is a high-level, interpreted scripting language used to create dynamic and interactive web pages. It was developed by Brendan Eich in 1995. JavaScript runs inside the browser and can also be used on the server side using environments like Node.js.

JavaScript is one of the core technologies of the web, along with HTML and CSS.

2. History of JavaScript

  • 1995 – Created by Brendan Eich
  • 1997 – ECMAScript standard introduced
  • 2009 – Node.js released
  • 2015 – ES6 introduced modern JavaScript

3. Features of JavaScript

  • Lightweight and fast
  • Interpreted language
  • Event-driven
  • Platform independent
  • Supports object-oriented programming
  • Runs in browser and server

4. JavaScript Execution

JavaScript code is executed by the browser’s JavaScript engine (Chrome uses V8 engine).

5. Basic JavaScript Program

console.log("Hello, JavaScript");

6. JavaScript Syntax

JavaScript syntax is similar to C and Java.

let a = 10; let b = 20; console.log(a + b);

7. Variables

Variables are used to store data values.

  • var – function scoped
  • let – block scoped
  • const – constant value
let name = "SkillSwap"; const pi = 3.14;

8. Data Types

  • Number
  • String
  • Boolean
  • Undefined
  • Null
  • Object
  • Array

9. Operators

  • Arithmetic Operators
  • Comparison Operators
  • Logical Operators
  • Assignment Operators

10. Conditional Statements

let age = 18; if (age >= 18) { console.log("Eligible to vote"); } else { console.log("Not eligible"); }

11. Loops

For Loop

for (let i = 0; i < 5; i++) { console.log(i); }

While Loop

let i = 0; while (i < 5) { console.log(i); i++; }

12. Functions

Functions are reusable blocks of code.

function add(a, b) { return a + b; }

13. Arrow Functions

const add = (a, b) => a + b;

14. Arrays

Arrays store multiple values in a single variable.

let fruits = ["apple", "banana", "orange"];

15. Objects

Objects store data in key-value pairs.

let student = { name: "John", age: 20 };

16. String Methods

  • length
  • toUpperCase()
  • toLowerCase()
  • slice()

17. Array Methods

  • push()
  • pop()
  • map()
  • filter()
  • forEach()

18. DOM (Document Object Model)

DOM allows JavaScript to interact with HTML elements.

document.getElementById("demo").innerText = "Hello DOM";

19. Events

Events respond to user actions like click, input, hover.

button.addEventListener("click", function() { alert("Button clicked"); });

20. Form Validation

JavaScript can validate user input before submitting forms.

21. Error Handling

try { let x = y + 10; } catch (error) { console.log(error); }

22. JSON

JSON is used for data exchange.

let data = { "name": "SkillSwap", "type": "Learning Platform" };

23. JavaScript in Web Development

  • Form validation
  • Dynamic content
  • API calls
  • Animations

24. Advantages of JavaScript

  • Runs in browser
  • Fast execution
  • Huge ecosystem

25. Disadvantages of JavaScript

  • Security risks
  • Browser compatibility issues

26. Applications of JavaScript

  • Web applications
  • Mobile apps
  • Server-side apps
  • Game development

27. Summary

JavaScript is a powerful scripting language that enables interactive and dynamic web experiences and is essential for modern web development.
