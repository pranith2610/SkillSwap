C Programming – Detailed Notes

1. Introduction to C

C is a general-purpose, procedural programming language developed by Dennis Ritchie in 1972 at Bell Laboratories. It is known for its efficiency and low-level memory access.

C is widely used in operating systems, embedded systems, compilers, and system-level programming.

2. History of C

  • 1969 – UNIX development started
  • 1972 – C language developed
  • 1978 – K&R C published
  • 1989 – ANSI C standard

3. Features of C

  • Simple and efficient
  • Procedural language
  • Fast execution
  • Low-level memory access
  • Portable
  • Rich set of operators

4. Structure of a C Program

A C program consists of:

  • Header files
  • Main function
  • Variables
  • Statements

5. Basic C Program

#include <stdio.h> int main() { printf("Hello, C!"); return 0; }

6. Keywords in C

Keywords are reserved words with special meaning.

  • int
  • float
  • char
  • if
  • else
  • while
  • for
  • return

7. Data Types in C

Basic Data Types

  • int
  • float
  • double
  • char

Derived Data Types

  • Array
  • Pointer
  • Structure
  • Union

8. Variables

Variables are used to store data values.

int a = 10; float b = 5.5;

9. Operators

  • Arithmetic Operators
  • Relational Operators
  • Logical Operators
  • Assignment Operators

10. Control Statements

Decision Making

if (a > b) { printf("A is greater"); }

Looping Statements

for(int i = 0; i < 5; i++) { printf("%d", i); }

11. Arrays

Arrays store multiple values of the same type.

int arr[5] = {1,2,3,4,5};

12. Functions

Functions are reusable blocks of code.

int add(int a, int b) { return a + b; }

13. Pointers

Pointers store memory addresses.

int a = 10; int *p = &a;

14. Structures

Structures group variables of different data types.

struct Student { int id; char name[20]; };

15. Unions

Unions store different data types in the same memory location.

16. File Handling in C

C supports file handling using FILE pointers.

FILE *fp; fp = fopen("data.txt", "r");

17. Advantages of C

  • High performance
  • Close to hardware
  • Wide usage

18. Disadvantages of C

  • No object-oriented support
  • No garbage collection

19. Applications of C

  • Operating Systems
  • Embedded Systems
  • Compilers
  • Device Drivers

20. Summary

C is a powerful and efficient programming language that forms the foundation for many modern programming languages.
