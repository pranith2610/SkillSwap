SQL – Structured Query Language (Detailed Notes)

1. Introduction to SQL

SQL stands for Structured Query Language. It is used to store, retrieve, manage, and manipulate data in relational database management systems (RDBMS). SQL was developed by IBM in the 1970s and later standardized by ANSI.

SQL is used with databases such as MySQL, PostgreSQL, Oracle, SQL Server, and SQLite.

2. Database and DBMS

A database is an organized collection of data. DBMS (Database Management System) is software that manages databases.

Examples of DBMS

  • MySQL
  • Oracle
  • PostgreSQL
  • SQL Server

3. Characteristics of SQL

  • Easy to learn
  • Standardized language
  • Supports large databases
  • Secure
  • Efficient data handling

4. SQL Data Types

Numeric Data Types

  • INT
  • FLOAT
  • DOUBLE

Character Data Types

  • CHAR
  • VARCHAR
  • TEXT

Date & Time Data Types

  • DATE
  • TIME
  • DATETIME

5. SQL Commands

SQL commands are categorized into five types.

DDL – Data Definition Language

  • CREATE
  • ALTER
  • DROP
  • TRUNCATE

DML – Data Manipulation Language

  • INSERT
  • UPDATE
  • DELETE

DQL – Data Query Language

  • SELECT

DCL – Data Control Language

  • GRANT
  • REVOKE

TCL – Transaction Control Language

  • COMMIT
  • ROLLBACK
  • SAVEPOINT

6. Creating a Table

CREATE TABLE students ( id INT, name VARCHAR(50), age INT );

7. Inserting Data

INSERT INTO students VALUES (1, 'John', 20);

8. Selecting Data

SELECT * FROM students;

9. WHERE Clause

Filters records based on conditions.

SELECT * FROM students WHERE age > 18;

10. UPDATE Statement

UPDATE students SET age = 21 WHERE id = 1;

11. DELETE Statement

DELETE FROM students WHERE id = 1;

12. Constraints

  • PRIMARY KEY
  • FOREIGN KEY
  • UNIQUE
  • NOT NULL
  • CHECK

13. Primary Key

Uniquely identifies each record in a table.

14. Foreign Key

Establishes relationship between tables.

15. Aggregate Functions

  • COUNT()
  • SUM()
  • AVG()
  • MAX()
  • MIN()

16. GROUP BY Clause

SELECT age, COUNT(*) FROM students GROUP BY age;

17. HAVING Clause

SELECT age, COUNT(*) FROM students GROUP BY age HAVING COUNT(*) > 1;

18. ORDER BY Clause

SELECT * FROM students ORDER BY age DESC;

19. Joins

Joins combine rows from multiple tables.

  • INNER JOIN
  • LEFT JOIN
  • RIGHT JOIN
  • FULL JOIN

20. Subqueries

SELECT * FROM students WHERE age > (SELECT AVG(age) FROM students);

21. Views

A view is a virtual table.

CREATE VIEW student_view AS SELECT name, age FROM students;

22. Indexes

Indexes improve query performance.

23. Transactions

Transactions ensure data integrity.

24. Advantages of SQL

  • Fast data retrieval
  • Secure access
  • Standard language

25. Disadvantages of SQL

  • Complex queries can be difficult
  • Cost of commercial DBMS

26. Applications of SQL

  • Web applications
  • Banking systems
  • E-commerce platforms
  • Data analytics

27. Summary

SQL is a powerful language for managing and querying relational databases. It is essential for backend development and data-driven applications.
