Java Programming – Detailed Notes

1. Introduction to Java

Java is a high-level, object-oriented programming language developed by Sun Microsystems and released in 1995. It was designed with the philosophy of Write Once, Run Anywhere (WORA), meaning compiled Java code can run on all platforms that support Java without recompilation.

Java is widely used in enterprise applications, mobile applications (Android), web servers, desktop applications, and embedded systems.

2. History of Java

Java was originally developed by James Gosling and his team at Sun Microsystems. Initially named Oak, it was later renamed Java.

  • 1991 – Project started (Oak)
  • 1995 – Java officially released
  • 2009 – Oracle acquired Sun Microsystems
  • Present – Java continues as a leading programming language

3. Features of Java

  • Simple
  • Object-Oriented
  • Platform Independent
  • Secure
  • Robust
  • Multithreaded
  • Portable
  • High Performance
  • Distributed

4. Java Architecture

Java works using the following components:

  • Java Source Code (.java)
  • Java Compiler (javac)
  • Bytecode (.class)
  • Java Virtual Machine (JVM)

5. JVM, JRE, JDK

JVM (Java Virtual Machine): Executes Java bytecode and provides platform independence.

JRE (Java Runtime Environment): JVM + core libraries.

JDK (Java Development Kit): JRE + development tools like compiler and debugger.

6. Basic Java Program

class HelloWorld { public static void main(String[] args) { System.out.println("Hello, Java!"); } }

7. Java Tokens

  • Keywords
  • Identifiers
  • Literals
  • Operators
  • Separators

8. Data Types in Java

Java supports two types of data types:

Primitive Data Types

  • byte
  • short
  • int
  • long
  • float
  • double
  • char
  • boolean

Non-Primitive Data Types

  • String
  • Array
  • Class
  • Interface

9. Control Statements

Control statements control the flow of execution in a program.

Decision Making

  • if
  • if-else
  • switch

Looping Statements

  • for
  • while
  • do-while

10. Object-Oriented Programming Concepts

Class

A class is a blueprint from which objects are created.

Object

An object is an instance of a class.

Inheritance

Inheritance allows one class to acquire properties of another class.

Polymorphism

Polymorphism allows methods to perform different tasks based on input.

Encapsulation

Encapsulation binds data and methods into a single unit.

Abstraction

Abstraction hides implementation details and shows only functionality.

11. Exception Handling

Exceptions are runtime errors. Java provides a powerful mechanism to handle them.

try { int a = 10 / 0; } catch (ArithmeticException e) { System.out.println("Cannot divide by zero"); } finally { System.out.println("Execution completed"); }

12. Multithreading

Multithreading allows multiple threads to run concurrently.

13. File Handling

Java provides File, FileReader, FileWriter classes for file handling.

14. Java Collections Framework

  • List – ArrayList, LinkedList
  • Set – HashSet, TreeSet
  • Map – HashMap, TreeMap

15. Advantages of Java

  • Secure
  • Portable
  • Scalable
  • Large community

16. Disadvantages of Java

  • Slower than C/C++
  • Memory consumption

17. Applications of Java

  • Web Applications
  • Mobile Applications
  • Enterprise Software
  • Game Development

18. Summary

Java is a powerful, flexible, and widely-used programming language suitable for building scalable and secure applications.
