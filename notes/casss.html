CSS – Cascading Style Sheets (Detailed Notes)

1. Introduction to CSS

CSS stands for Cascading Style Sheets. It is used to control the appearance, layout, and design of web pages written in HTML. CSS separates content from presentation, making websites easier to maintain and more flexible.

CSS was introduced by the World Wide Web Consortium (W3C) in 1996.

2. Why CSS is Needed

  • Improves website appearance
  • Maintains consistency across pages
  • Reduces HTML complexity
  • Enables responsive design

3. Types of CSS

Inline CSS

Applied directly inside HTML tags.

<h1 style="color:red;">Hello</h1>

Internal CSS

Defined inside the <style> tag within the HTML file.

External CSS

Written in a separate .css file and linked to HTML.

<link rel="stylesheet" href="style.css">

4. CSS Syntax

selector { property: value; }

5. CSS Selectors

  • Universal Selector (*)
  • Element Selector
  • Class Selector (.class)
  • ID Selector (#id)
  • Group Selector

6. Colors in CSS

  • Color names
  • HEX values
  • RGB / RGBA
  • HSL
color: #ff0000; background-color: rgb(255,0,0);

7. Background Properties

  • background-color
  • background-image
  • background-repeat
  • background-position
  • background-size

8. Text Properties

  • color
  • text-align
  • text-decoration
  • text-transform
  • line-height

9. Font Properties

  • font-family
  • font-size
  • font-style
  • font-weight

10. Box Model

The CSS Box Model consists of:

  • Content
  • Padding
  • Border
  • Margin
div { margin: 10px; padding: 15px; border: 1px solid black; }

11. Display Property

  • block
  • inline
  • inline-block
  • none

12. Position Property

  • static
  • relative
  • absolute
  • fixed
  • sticky

13. Float and Clear

Used for positioning elements.

14. Flexbox

Flexbox is a one-dimensional layout system.

  • display: flex
  • justify-content
  • align-items
  • flex-direction

15. CSS Grid

Grid is a two-dimensional layout system.

display: grid; grid-template-columns: repeat(3, 1fr);

16. Responsive Design

Responsive design ensures websites work on all devices.

Media Queries

@media (max-width: 600px) { body { background-color: lightgray; } }

17. Pseudo-classes

  • :hover
  • :active
  • :focus
  • :visited

18. Pseudo-elements

  • ::before
  • ::after
  • ::first-letter

19. Transitions

Transitions add smooth animation effects.

button { transition: background 0.3s ease; }

20. Animations

CSS animations use keyframes.

@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } }

21. Z-index

Controls stacking order of elements.

22. Advantages of CSS

  • Improves page speed
  • Easy maintenance
  • Consistent design

23. Disadvantages of CSS

  • Browser compatibility issues
  • Complex layouts can be difficult

24. Applications of CSS

  • Website design
  • Mobile-first layouts
  • Animations
  • UI/UX design

25. Summary

CSS is a core web technology used to design attractive, responsive, and user-friendly websites.
