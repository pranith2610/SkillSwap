Python Programming – Detailed Notes

1. Introduction to Python

Python is a high-level, interpreted, general-purpose programming language created by Guido van Rossum and released in 1991. Python focuses on code readability and simplicity, making it one of the easiest languages to learn and use.

Python is widely used in web development, data science, artificial intelligence, machine learning, automation, and scripting.

2. History of Python

  • 1989 – Python development started
  • 1991 – Python first released
  • 2000 – Python 2.0 released
  • 2010 – Python 3.0 released
  • Present – Python dominates AI & Data Science

3. Features of Python

  • Easy to learn and use
  • Interpreted language
  • Dynamically typed
  • Object-oriented
  • Large standard library
  • Cross-platform
  • Open-source

4. Python Installation

Python can be downloaded from the official website: python.org. After installation, Python can be run using the terminal or command prompt.

5. First Python Program

print("Hello, Python!")

6. Python Keywords

Keywords are reserved words in Python.

  • if
  • else
  • elif
  • for
  • while
  • def
  • return
  • import

7. Variables in Python

Variables are used to store data values. Python does not require explicit variable declaration.

x = 10 name = "SkillSwap"

8. Data Types in Python

Built-in Data Types

  • int
  • float
  • complex
  • string
  • list
  • tuple
  • set
  • dictionary

9. Type Casting

x = int(10.5) y = str(25)

10. Operators

  • Arithmetic Operators
  • Comparison Operators
  • Logical Operators
  • Assignment Operators

11. Conditional Statements

age = 18 if age >= 18: print("Eligible to vote") else: print("Not eligible")

12. Loops

For Loop

for i in range(5): print(i)

While Loop

i = 0 while i < 5: print(i) i += 1

13. Functions

Functions are reusable blocks of code.

def add(a, b): return a + b

14. Lambda Functions

x = lambda a: a + 10 print(x(5))

15. Lists

Lists are ordered and mutable collections.

fruits = ["apple", "banana", "cherry"]

16. Tuples

Tuples are ordered but immutable collections.

numbers = (1, 2, 3)

17. Sets

Sets are unordered collections with unique values.

colors = {"red", "green", "blue"}

18. Dictionaries

Dictionaries store data in key-value pairs.

student = { "name": "John", "age": 20 }

19. File Handling

Python supports file handling for reading and writing files.

file = open("data.txt", "r") print(file.read()) file.close()

20. Exception Handling

try: x = 10 / 0 except: print("Error occurred") finally: print("Done")

21. Modules

Modules allow code reuse.

import math print(math.sqrt(16))

22. Object-Oriented Programming

Python supports OOP concepts like classes and objects.

class Student: def __init__(self, name): self.name = name def display(self): print(self.name)

23. Advantages of Python

  • Easy syntax
  • Huge community
  • Powerful libraries

24. Disadvantages of Python

  • Slower execution
  • Not ideal for mobile apps

25. Applications of Python

  • Web Development
  • Data Science
  • Machine Learning
  • Automation
  • Game Development

26. Summary

Python is a versatile and beginner-friendly programming language used across multiple domains. Its simplicity and power make it one of the most popular languages in the world.
